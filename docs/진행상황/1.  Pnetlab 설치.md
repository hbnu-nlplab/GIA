# 5. Putty 설치
![[IMG-20251122123543568.png]]
pnetlab 장비를 더블클릭하면 웹에서 터미널이 열리지만 붙혀넣기가 안되서 명령어 입력에 불편함이 있다.

현재 단계에서는 장비에 ssh 활성화를 안했기 때문에 ssh 접속이 불가능하다.

그래서 pnetlab 장비에 텍스트 복붙을 위해서 telnet으로 접속하면 복사-붙혀넣기 사용이 가능하다.

터미널에서 사용하거나 Putty로 접속한다.

- IP는 PnetlabVM의 IP 주소로 하면된다.
